Syntaxe

Child: >

nav>ul>li

<nav>
<ul>
<li></li>
</ul>
</nav>

Multiplications: *

ul>li.item$*5

<ul>
<li class="item1"></li>
<li class="item2"></li>
<li class="item3"></li>
<li class="item4"></li>
<li class="item5"></li>
</ul>

Sibling: +

div+p+bq

<div></div>
<p></p>
<blockquote></blockquote>

ID and CLASS attributes

#header

<div id="header"></div>

.title

<div class="title"></div>

form#search.wide

<form id="search" class="wide"></form>

p.class1.class2.class3

<p class="class1 class2 class3"></p>

Climb-up: ^

div+div>p>span+em^bq

<div></div>
<div>
<p><span></span><em></em></p>
<blockquote></blockquote>
</div>

div+div>p>span+em^^bq

<div></div>
<div>

</div>

Text: {}

a{Click me}

p>{Click }+a{here}+{ to continue}

<p>Click <a href="">here</a> to continue</p>
